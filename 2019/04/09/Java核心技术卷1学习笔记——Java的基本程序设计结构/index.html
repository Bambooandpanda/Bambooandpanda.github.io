Java核心技术 卷1 学习笔记

Java的基本程序设计结构

  • 当心注释中的\u，注释//\u00A0会替换为一个换行符。类似地，注释//Look inside C:\users 是一个语法错误，因为\u后面并未跟着4个16进制数

  • 布尔类型和其他类型之间不能相互转换。

  • 常量
    在Java中，利用关键字final指示常量。关键字final表示这个变量只能被赋值一次。一旦被赋值之后，就不能再被更改了。习惯上，常量都要大写
    如果希望某个常量可以在一个类中的多个类方法中使用，通常称这些常量为类常量

  • Math.floorMod方法主要是为了解决整数取余的问题。

  • Math.round方法可以对浮点值四舍五入。

  • Math类还提供了一些常用的三角函数，指数函数，反函数，对数函数等方法。

  • 不必在数学方法和常量名之前添加前缀Math,只要在源文件的顶部加入import static java.lang.Math.*

  • 数值之间的转换

  • 注意短路逻辑运算符和非短路逻辑运算符的区别

  • 位运算符注意>>>运算符会用0填充高位，与>>不同，它会用符号位填充高位。

  • Java不使用逗号运算符。不过，可以在for语句的第一部分和第三部分中使用逗号运算符分割表达式列表。

  • 枚举类型。

  • 字符串

    • 子串substring方法

    • 拼接 使用+号

    • 不可变字符串
      由于不能修改Java字符串中的字符，所以在Java文档中将String类对象称为不可变字符串，如同数字3永远是数字3一样，字符串“Hello”永远包含H e l l o代码单元序列。当然，可以修改字符串变量greeting，让它引用（注意Java的引用和C++的引用不同）另外一个字符串，这就如同可以将存放在3的数值变量改成存放4一样。

    • 检测字符串是否相等
      可以使用equals方法检测两个字符串是否相等。对于表达式s.equals(t)表示，如果s与t相等，返回true，否则返回false。注意，t可以是字符串，也可以是字符串字面量。
      千万不要使用==运算符检测两个字符串是否相等！这两个运算符只能够确定这两个字符串是否在同一个位置上（相当于C++里面char*指针指向的地址是不是一样的）

    • 检测字符串是否相等（忽略大小写） equalsIgnoreCase()

    • 空串与Null串
      空串是长度为0的字符串，它是一个Java对象，有自己的长度（0）和内容（空）
      String变量还可以存放一个特殊的值，名为null，表示目前没有任何对象与该变量关联
      检查一个字符串是否为空 str.length()或者str.equals(“”)
      检查一个字符串是否为null if(str==null)

    • 码点与代码单元（留坑，以后补）

    • 字符串比较 int compareTo(String other)

    • 判断是否相等boolean equals(Object other)

    • 判断是否相等（忽略大小写）boolean equalsIgnoreCase(String other)

    • 判断是否以str开头或者结尾boolean startsWiths(String str) boolean endsWith(String str)

    • 返回字符串的长度int length()

    • 返回一个新的字符串，这个字符串用newString 代替原始字符串中的所有oldString
      String replace(Charsequence oldString,Charsequence newString)

    • 将大写改为小写（或者小写改成大写）String toLowerCase() String toUpperCase()

    • 删除字符串开头和结尾的空格 String trim()

    • 返回一个新的字符串，用给定的定界符连接所有的元素String join(CharSequence delimiter,CharSequence… elements)

    • 构建字符串
      如果需要用许多小段的字符串构建一个字符串，那么可以按照下列步骤进行。首先，构造一个空的字符串构造器。

      每次需要添加一部分内容时，就调用append方法

      (效率String<StringBuffer<StringBuilder)

      • 构造一个空的字符串构建器StringBuilder()
      • 返回构建器或者缓冲器中的代码单元数量int length()
      • 追加一个代码单元并返回this StringBuilder append(char c) StringBuilder append(String str)
      • 将第i个代码单元设置为c void setCharAt(int i,char c)
      • 在offset位置插入一个字符串或者代码单元并返回this StringBuilder insert(int offset,String str或者char c)
      • 返回一个与构建器或者缓冲器内容相同的字符串String toString()

    • 输入输出

      • 读取输入
        想要通过控制台进行输入，首先需要构造一个Scanner对象，并与标准输入流System.in关联

        现在，就可以使用Scanner的各种方法实现输入输出操作了。Scanner类定义在java.util包中，当使用的类不是定义在基本java lang包中时，一定要使用import指示关键字将相应的包加载进来

        • 读取整数 in.nextInt()
        • 读取下一行内容in.nextLine()
        • 读取下一个单词in.next()
        • 用给定的输入流创建一个Scanner对象Scanner(InputStream in)
        • 读取并转换下一个表示整数或者浮点数的字符序列double nextDouble()
        • 检测输入中是否还有其他单词boolean hasNext() boolean hasNextInt() boolean hasNextDouble()

      • 格式化输出
        可以使用类似于C语言的System.printf()方法

      • 文件的输入输出
        想要对文件进行读取，就需要一个File对象构造一个Scanner对象

        如果文件名中包含反斜杠符号，就要记住在每个反斜杠之前加额外的反斜杠

        想要写入文件，就需要构造一个PrintWriter对象，在构造器中，只需要提供文件名

        如果文件不存在，创建该文件，可以像输出到System.out一样使用print println 以及 printf

        • 构造一个从给定文件读取数据的Scanner Scanner(File f)
        • 构造一个从给定字符串读取数据的Scanner Scanner(String Data)
        • 构造一个将数据写入文件的PrintWriter PrintWriter(String fileName)
        • 根据给定的路径名构造一个Path static Path get(String pathName)

    • 控制流程

      • 在循环中，检测两个浮点数是否相等需要格外小心，因为循环可能永远不会结束，由于舍入的误差，最终可能得不到精确值。

      • switch 的 case 标签可以是

        • char byte short 或者int的常量表达式
        • 枚举常量
        • 字符串字面量

      • 带标签的continue 和 break语句

      • Java中没有提供运算符重载功能

      • 大数值（在java.math包中）

        • BigInteger可以实现任意精度的整数运算

        • BigDecimal实现了任意精度的浮点数运算

        • 可以使用静态的valueOf方法将普通的数值转换为大数值

        • BigInteger

          • BigInteger add(BigInteger other)
          • BigInteger subtract(BigInteger other)
          • BigInteger multiply(BigInteger other)
          • BigInteger divdie(BigInteger other)
          • BigInteger mod(BigInteger other)
          • BigInteger compareTo(BigInteger other)
          • static BigInteger valueOf(long x)

        • BigDecrimal与之类似

      • 数组

        • for each循环

        • 匿名数组
          注意，在使用匿名数组的时候，不需要new

        • 数组的拷贝
          如果希望将一个数组的所有值全都拷贝到一个新的数组中，就要使用Arrays类的copyOf方法
          int[] copiedLuckyNumbers=Arrays.copyOf(luckyNumbers,luckyNumbers.length),第二个参数是新数组的长度。这个方法通常用来增加数组的大小。

        • 数组的排序
          想要对数值型数组进行排序，可以使用Arrays类的sort方法：

        • Math.random方法将返回一个0到1之间的随机浮点数（包含0不包含1）用n乘以这个浮点数，就可以得到从0到n-1的一个随机数
